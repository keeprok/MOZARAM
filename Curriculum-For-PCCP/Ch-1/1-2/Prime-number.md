# 소수 판별 (에라토스테네스의 체)

## 요약 설명

- 에라토스테네스의 체 는 1부터 주어진 숫자의 범위내에서 소수 를 찾는 알고리즘입니다
  - 소수 : 1과 자기 자신 외에 약수가 없는 수
  - 시간 복잡도 : O(N log(logN))
    <br/>

## 언제 사용하면 좋은가?

- 범위 내 모든 소수를 구할 때
- 소수를 많이 다루는 문제를 풀 때
