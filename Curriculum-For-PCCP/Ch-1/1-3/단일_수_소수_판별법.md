# {알고리즘/자료구조}

## 요약 설명

- 단일 수 소수 판별법은 특정한 하나의 정수가 소수인지 여부를 확인하는 방법입니다. 이 알고리즘을 사용하면 단일 정수가 소수인지 빠르게 판별할 수 있습니다.

<br/>

## 언제 사용하면 좋은가?

- 특정한 하나의 수가 소수인지 확인해야 할 때
- 소수를 이용하는 알고리즘의 초기 조건을 확인할 때
  <br/>
