# 01. 자료구조(Data Structure)

- 데이터를 효율적으로 다룰수 잇게 도와주는 데이터 보관방법 , 데이터에 관한 연산의 총제
- 자료구조의 형식
  - 단순 자료구조
  - 복합 자료 구조
- 단순 자료 구조
  - int , long ,double
- 복합 자료 구조
  - 선형(Linear)
    - 배열 (Array)
    - 스택 (Stack)
    - 큐 (Queue)
    - 힙 (Heap)
    - 연결 리스트 (Linked List)
